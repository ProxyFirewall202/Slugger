# Slugger
Slugger v1.4.7 beta
+Minor visual bug fix

Slugger is a fast-paced top-down bullet-hell shooter full of eternal action and fun!

The game's core are waves made of "Hivoxes", which in turn, spawn enemies that you'll need to defeat if you wish to proceed to the following wave. Some waves contain bosses. 

Once you perish, you may upgrade your slug in the Skill Tree, and become more powerful, so you can get to harder waves. 

Once everything in the Skill Tree is maxed out, there will be an *Ancient* surprise waiting for you... Maybe you can use this power to become even more powerful... and defeat the final boss...

This game is mainly only supported on PC and some types of Chromebooks. It half-supports mobile, but is quite hard to use on mobile and may not even run.
